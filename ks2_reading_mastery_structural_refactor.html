Reading settings

Scaffold mode is for teaching, not test fidelity. Punctuation is treated here as a meaning-and-prosody support strand, not a separate bag of SPaG tricks.

Optional AI

In a single local HTML file, any direct API key use is client-side. Session-only storage is the safer default. Some providers may still reject direct browser calls because of CORS or auth rules. AI sets in this version go through a stricter local validator: only allowed question types are accepted, evidence quotes must exist in the passage, and leaky hints are stripped or rejected. The core app still works fully with AI turned off.

How this app is designed

The loop is simple on purpose: read a real passage, answer without answer leakage, get focused feedback after effort, then see weak skills return later in mixed review. Smart review weights weak content domains, overdue review, under-practised passage types and recent misses so problems reappear before they fade. This version also adds a tighter follow-up pipeline: local deterministic enrichment sets from the answered question, plus optional AI-generated sets that are only accepted if they pass a stricter local validator.
