Optional AI

In a single local HTML file, any direct API key use is client-side. Session-only storage is the safer default. Some providers may also reject direct browser requests or hit CORS restrictions. The app still works fully without AI, and the provider adapter is isolated so you can later swap in a tiny proxy if needed.

How this app is designed

This tool is built around the real job of KS2 grammar mastery: recognise what a sentence is doing, discriminate between closely related grammar ideas, correct errors, manipulate clauses and phrases, and apply that knowledge under KS2 GPS-style conditions.
