Import / export / reset

Progress is stored in your browser with localStorage. Export regularly if you care about keeping the data.

Reset removes progress but keeps the app file itself. Export first if you may want the data later.

Practice settings

True Test Mode never shows early hints or worked support before marking. Read-aloud is lightweight browser TTS for accessibility only.

Optional AI

In a single local HTML file, any direct API key use is client-side. Session-only storage is the safer default. Some providers may reject direct browser requests or hit CORS restrictions. The arithmetic engine still works fully without AI, and the provider adapter is isolated so you can later swap in a tiny proxy if needed.

How this trainer is designed

This tool is built around the real job of KS2 arithmetic: recall number facts quickly, carry written methods accurately, keep place value under control, and stay reliable under time pressure. The app separates learning mode from test mode on purpose, because the best way to build fluency is not identical to the best way to measure SATs readiness.

Core learning loop

Genuine first attempt No answer leakage first Quick correction after the attempt Worked help only when needed Fresh retry later with spacing
Honest limitation: the app mirrors the released arithmetic paper closely on timing and question style, but it does not reliably award official handwritten method marks on long multiplication and long division. It auto-scores the final answer locally and leaves method quality for self- or parent review.
