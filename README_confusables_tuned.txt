KS2 Spelling Sprint — confusables / homophones tuned pack

Summary
- Base: prior polished sentence bank
- Focus of this pass: highest-stakes confusable and near-homophone families for KS2 dictation
- Words hand-tuned: breath, breathe, heard, heart, eight, eighth, though, although,
  thought, through, thorough, woman, women, conscience, conscious, reign, weight, queue
- Files included: preview.html plus sentence-bank-01.js to sentence-bank-06.js

What changed
- Rewrote the target families by hand instead of keeping generic contextual lines.
- Strengthened meaning cues so the dictated sentence makes the target word more obvious.
- Kept sentences short, natural, and read-aloud friendly.
- Favoured strong noun / verb / singular / plural clues where that helps:
  - breath vs breathe
  - heard vs heart
  - eight vs eighth
  - woman vs women
  - conscience vs conscious
  - though / although / thought / through / thorough
  - reign
  - weight
  - queue
- Avoided meta-practice filler and avoided weak “school worksheet” style lines.

Notes
- The app rotates all available sentence variants for each word, so targeted quality matters.
- This pack is safe to use as a drop-in replacement for the six sentence-bank files.
- preview.html is included only for convenience; if your current HTML has newer code changes,
  you can keep your own preview.html and replace just the six sentence-bank files.
